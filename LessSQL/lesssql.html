LessSQL Database Client

  • LessSQL is a database client for MySQL database server, it is a smart, intuitive and easy-to-use software which helps institutions like schools, library, hospitals, etc. in constant data query, manipulation and updatation.
  • It helps you to think less about SQL and more about data.
  • It is an open-source and multiplatform software.
  • Supports both Windows and Linux OS.
